# CJ Eclipsed Portfolio

This portfolio website showcases esports and sports event photography, including featured event bundles such as SparkOut, Motorsport UK, and NSE Finals.

I am Connor Atherley, an esports production student at Nottingham Trent University with a passion for capturing live events, behind-the-scenes production, and the energy of competition. I work across photography, videography, and event storytelling, helping brands and organisers document the moments that matter most.

This portrait reflects the person behind the lens — someone dedicated to creating visual stories that feel immersive, honest, and memorable.
